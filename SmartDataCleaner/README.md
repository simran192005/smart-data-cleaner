# Smart Data Cleaner & Visualizer

A Streamlit-based web application that automatically cleans messy datasets, analyzes data quality, detects outliers, generates insights, and provides interactive visualizations.

The goal of this project is to make dataset exploration easier by automating common data preprocessing and analysis tasks.

---

## Features

- Upload CSV or Excel datasets
- Automatic dataset cleaning
- Data quality score calculation
- Interactive visualizations
- Correlation heatmap
- Automatic data insights
- Outlier detection
- Download cleaned dataset

---

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly

---

## How to Run the Project

### 1 Install dependencies

```
pip install -r requirements.txt
```

### 2 Run the app

```
streamlit run app.py
```
